# 前端说明文档

### 一、项目概述

​       本项目是一款面向爱宠人士的宠物社交服务平台，该部分为平台的前端部分，主要负责实现用户界面展示、用户交互以及与后端接口的数据交互，展示内容包括宠物动态分享、社交互动以及多项宠物服务提供等。前端采用 **Vue2 + Element UI** 构建响应式 Web 应用，实现 PC 端和移动端浏览器访问。

### 二、模块功能介绍

##### 2.1用户模块

功能介绍：

- 用户的登陆注册，存储用户个人信息，同时包括资料管理和个性化设置
- 用户关注、收藏、评论、点赞等行为支持
- 用户的消息通知模块

##### 2.2 社交动态模块

功能介绍：

- 动态发布，可以添加文字、图片和话题标签等，方便话题分类和推荐
- 动态列表展示，还可以按照发布时间、热度等排序
- 社交互动，可以收藏、点赞帖子，关注作者，评论互动等等

##### 2.3宠物服务模块

功能介绍：

- 提供多项宠物服务，如宠物美容、上门遛狗、临时寄养、宠物托管、紧急救助、就近宠物医院预约等
- 多家商户列表展示，及详情提供
- 在线预约、联系，订单管理等

##### 2.4搜索与分类模块

功能介绍：

- 全局搜索，根据关键词搜索用户、商户等
- 根据标签分类，快速获取感兴趣的帖子内容等

### 四、技术选型

| 技术          | 作用                          |
| ------------- | ----------------------------- |
| Vue2          | 核心框架                      |
| Vue Router    | 路由管理                      |
| Axios         | HTTP 请求库，用于调用后端 API |
| Element UI    | UI 组件库                     |
| Vue CLI       | 项目构建工具                  |
| Node.js + npm | 依赖管理与运行环境            |

### 五、目录结构（大致规划）

```
src/
|-assets/       #放置图片等静态资源
|-components/   #放置通用组件
   |-status/    #状态组件（点赞、收藏、关注）
   ···
|-router/       #路由
   |-index.js   #路由设置
|-views/        #页面
   |-user/      #用户相关页面
   |-social/    #社交相关页面
   |-service/   #服务相关页面
   |-search/    #搜索相关页面
|-App.vue       #根组件
|-main.js       #全局设置

```



### 六、运行方式

##### 1、安装依赖

```
npm install
```

##### 2、启动

```
npm run serve
```

用浏览器访问运行地址

##### 3、打包

```
 npm run build
```
