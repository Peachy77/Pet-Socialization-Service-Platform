### 项目简介

本项目旨在开发一个养宠人士社交平台，主要功能聚焦在社交活动和提供服务，前者包括养宠人士的动态发布和分享，用户可发布宠物照片、文字日记，支持话题标签，方便进行养宠心得分享和同好社交；后者包括宠物美容、上门遛狗、临时寄养、宠物托管、紧急救助等多项服务接入，通过与商户合作方式，构建一个完善的宠物社交服务平台。
